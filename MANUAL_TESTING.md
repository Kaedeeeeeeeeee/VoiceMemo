# VoiceMemo 手动测试清单

## 1. 录音功能（Home Tab）

| # | 操作 | 预期结果 |
|---|------|---------|
| 1.1 | 点击麦克风按钮 | 波形动画出现，计时器开始，状态显示"录音中" |
| 1.2 | 录音中点击暂停 | 波形停止，状态变"已暂停"，计时器暂停 |
| 1.3 | 暂停后点击继续 | 波形恢复，计时器继续 |
| 1.4 | 录音中点击书签按钮 | 弹出 AddMarkerSheet，可输入文字/拍照 |
| 1.5 | 长按停止按钮 1.5 秒 | 进度环填满，震动反馈，录音停止并保存 |
| 1.6 | 停止后 | 录音出现在历史列表，自动开始转写（看到"正在转写"） |
| 1.7 | 点击"导入音频" | 文件选择器出现，选 .m4a 文件后创建新录音 |
