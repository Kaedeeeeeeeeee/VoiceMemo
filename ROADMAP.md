# 产品路线图：差异化定位与功能优先级

## 定位宣言

**纯软件 AI 语音笔记，覆盖 iPhone + Apple Watch + Mac，零硬件成本，面向国际市场。**

对标 Plaud（$159硬件 + $80-240/年订阅），我们用纯软件方案提供同等甚至更好的体验，价格只有其 1/3。对标闪念贝壳，我们在长录音场景（会议/课程/访谈）上做得更深，覆盖国际市场。

---

- [x] iPhone/Watch/Mac 三端录音
- [x] Watch 录音 + WatchConnectivity 同步到 iPhone

- [x] GPT-4o 转写润色、自动标题

## Phase 0：国际化基础（1-2 周）

> 目标：让产品可以面向英语市场发布

### 0.1 完善英文本地化
- [ ] 补全所有 UI 字符串的英文翻译（当前只有部分）
- [ ] 转写语言从硬编码 `zh` 改为自动检测或用户选择
- [ ] 摘要模板的 system prompt 支持英文输出（当前全中文）
- [ ] AI 对话的 system prompt 跟随用户语言

### 0.2 国际定价
- [ ] 增加 USD 定价产品（建议 $4.99/月，$39.99/年）
- [ ] StoreKit 配置支持多币种
- [ ] 定价策略：比 Plaud Pro（$79-99/年）便宜一半，比 Otter Pro（$99/年）也便宜

### 0.3 试用策略调整
- [ ] 当前：1 次免费试用录音 → 改为：每月 3 次免费 AI 处理（转写+摘要）
- [ ] 录音功能本身不限制（录音不消耗付费额度，AI 处理才消耗）
- [ ] 目的：让用户充分体验价值再决定付费，对标 Plaud 免费 300 分钟/月

## Phase 1：核心竞争力（3-4 周）

> 目标：在长录音 + Watch 体验上建立差异化优势

### 1.1 Watch 电话录音 📞
- [ ] 检测 iPhone 通话状态（CallKit / WatchConnectivity）
- [ ] 通话时自动或手动在 Watch 端启动录音
- [ ] Watch 麦克风离嘴巴近，可捕捉用户声音；免提/音量较大时可录双方
- [ ] 录音结束后自动同步到 iPhone 进行转写
- [ ] **定位：Plaud Note 骨传导的软件替代方案，零硬件成本**

### 1.2 多语言转写
- [ ] AssemblyAI 支持多语言，改为自动语言检测（`language_detection: true`）
- [ ] 或让用户在设置中选择常用语言
- [ ] 支持中英混合转写场景

### 1.3 长录音体验优化
- [ ] 转写进度指示（当前大文件分片后用户不知道进度）
- [ ] 转写结果分段显示 + 时间戳锚点（点击段落跳转音频位置）
- [ ] 长录音摘要分章节输出

### 1.4 更多摘要模板
- [ ] 课堂笔记模板（知识点 + 重点 + 课后任务）
- [ ] 播客/访谈模板（嘉宾观点 + 金句 + 话题索引）
- [ ] 头脑风暴模板（想法分组 + 可行性评估）
- [ ] 用户自定义模板（自己写 prompt）

---

## Phase 2：导出生态 + 增长（2-3 周）

> 目标：打通与主流工具的连接，降低用户迁移成本

### 2.1 导出格式扩展
- [ ] Markdown 文件导出（.md）
- [ ] 纯文本导出（.txt）
- [ ] Word 文档导出（.docx）—— 商务场景刚需
- [ ] 音频 + 转写打包导出（.zip）

### 2.2 第三方工具集成（按优先级）
- [ ] **Notion**：OAuth + Create Page API，一键将摘要推送到指定 database
- [ ] **Apple 备忘录/提醒事项**：通过 EventKit / 系统 API 直接写入
- [ ] **Bear / Obsidian**：URL Scheme（`bear://x-callback-url/create`）/ 导出到 vault 目录
- [ ] **Things / OmniFocus**：URL Scheme 导入待办项（从 Action Items 模板直出）
- [ ] **Google Docs**：API 集成（国际市场刚需）

### 2.3 分享与传播
- [ ] 摘要生成精美图片卡片（适合社交分享）
- [ ] Widgets 优化（iOS 锁屏/桌面小组件快速开始录音）
- [ ] Siri / Shortcuts 集成（"Hey Siri, start a meeting recording"）
- [ ] Apple Watch Complication 一键录音

---

## Phase 3：高级差异化（4-6 周）

> 目标：建立技术壁垒和用户粘性

### 3.1 智能知识库
- [ ] 跨录音搜索（全文检索 + 语义搜索）
- [ ] "Ask AI" 基于所有历史录音回答问题（RAG）
- [ ] 录音自动打标签 + 智能分类
- [ ] 录音间的关联推荐（"这次会议提到了上周讨论的 X 议题"）

### 3.2 实时转写（Live Transcription）
- [ ] 录音过程中实时显示文字（Apple Speech Framework 本地识别）
- [ ] 录音结束后再用 AssemblyAI 做高精度转写替换
- [ ] Watch 端也展示实时文字

### 3.3 团队协作（未来）
- [ ] 录音分享链接
- [ ] 协作编辑转写稿
- [ ] 团队版订阅

### 3.4 声纹识别（Speaker Voiceprint）
- [ ] 学习常见说话人的声纹
- [ ] 新录音自动匹配已知说话人（不再是"说话人A"而是"张三"）

## 定价策略对比

| 维度 | Plaud | 闪念贝壳 | 我们（建议） |
|------|-------|---------|------------|
| 硬件成本 | $159-169 | $0 | **$0** |
| 月付 | — | ¥12/月 | **$4.99/月** |
| 年付 | $79-240/年 | ¥128/年 | **$39.99/年** |
| 免费额度 | 300分钟/月（但需硬件） | 1分钟录音 + 3万点数 | **不限录音 + 3次AI/月** |
| 第一年总成本 | **$239-409** | **~$20** | **$0-50** |

**关键策略：录音永远免费，AI 处理按次/按量收费。** 让用户积累大量录音后产生转写需求，自然转化付费。
